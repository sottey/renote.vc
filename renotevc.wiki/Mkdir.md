**Mkdir** takes one argument, if not provided it'd ask you in any way via TUI prompt.

- **Usage**: `renotevc mkdir [flags]`
- **Aliases**: `mkdir`, `md`
- **Flags**: `-h`, `--help` -> help for mkdir
