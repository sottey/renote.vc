**List command is an alternative shortcut of the unix's `ls` command. It logs a list of all nodes under the defined note path.**

- **Usage**: `renotevc list [flags]`
- **Aliases**: `list`, `ls`
- **Flags**: `-h`, `--help` -> help for list
