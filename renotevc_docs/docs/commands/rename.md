# deletenote command
Delete a note from a todo by specifying a todoid and a noteid

## Usage
`redovc deletenote <todoID> <noteID> [flags]`

## Aliases
  deletenote, dn

## Examples
#### To see your todos with notes:
`redovc list --notes`

#### To delete note 0 from todo 3:
`redovc dn 3 0`

## Flags
`-h, --help   help for deletenote`
